# Task Log Reference Lists

Fixed lookup lists used by `workflows/daily-planning.md` when filling each
task's row in `workdone/<Mon-YY>.md`. Category and AppID are independent
picks per task — not a fixed pairing — so any category can occur with any
AppID depending on what the task is actually for.

## Category

- Regular support
- New Initiatives
- Migration Activities
- Meetings
- Automation
- Production Calls
- Production Preparedness
- Info for Management
- Mandatory Trainings
- Patching Activities
- Debugging Sessions
- Adhoc Request
- KT sessions
- Tachyon Infra Activities
- Miscellaneous

## Status

(For consistency with the user's tracking tool — not a column in the
monthly hours log itself.)

- Not Started
- In-Progress
- Blocked
- Completed
